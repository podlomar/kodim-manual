# Struktura kurzu

Kurz je vždy reprezentovaný složkou. Soubor `entry.yml` má následující strukturu.

```
title: Název kurzu
lead: Krátký popisek obsahu kurzu, jedna dvě věty
lessons:
  - link-na-lekci-1
  - link-na-dalsi-lekci
  - posledni-lekce
```

## Git repozitář

Složka kurzu můžu být Git repozitářem.
